Section 2

Methodology

VXIS conducted this assessment using a combination of automated scanning and manual verification techniques. The methodology follows industry-standard frameworks including OWASP Testing Guide (OTGv4), PTES (Penetration Testing Execution Standard), and NIST SP 800-115. Automated tools were used for initial discovery and coverage, with manual analysis applied to validate findings, eliminate false positives, and identify logic-level vulnerabilities that automated tooling cannot detect.
